# 🦟 Mosquito Life — 蚊の一生

蚊の視点で人間の世界を生き延びる **3Dサバイバルゲーム**。
人にバレないように飛び、隠れ、服に止まり、スキを突いて吸血する。叩かれたら一発アウト。

> Vite + Three.js 製。画像・音声などの外部アセットは一切なし（すべて手続き生成）。
> PC（キーボード＋マウス）とスマホ横画面（仮想ジョイスティック）の両方に対応。

---

## 🎮 遊び方

目標は「**できるだけ長く生き延びてスコアを稼ぐ**」こと。お腹（満腹ゲージ）が
ゼロになると餓死、人間・ネコに叩かれたり、煙にやられても死ぬ。

### 基本ループ
1. 人間に**そっと近づく**（速く飛ぶと羽音でバレる）
2. 肌（腕・首・顔・足首）に**着地して吸血** → 満腹＆スコアUP
3. バレそうになったら**服・家具・天井・観葉植物に止まって隠れる**（スタミナ回復・警戒度ダウン）
4. 満腹になったら**産卵**してボーナス
5. ミッションをこなして **WAVE** を進めるほど難易度UP＆高得点

### 操作

| | PC | スマホ（横画面推奨） |
|---|---|---|
| 移動 | `WASD` / 矢印キー | 左ジョイスティック |
| 視点 | マウス | 画面右半分をドラッグ |
| 上昇/下降 | `Space` / `Shift` | `▲` / `▼` ボタン |
| 吸血・着地・離陸 | 左クリック / `E` | 🩸 ボタン |
| ダッシュ | `Ctrl` | 💨 ボタン |

## ⚠️ 脅威

- **人間** … 近い・羽音が大きい・吸血されていると気づき、警戒度が上がると手で叩いてくる
- **ネコ** … 低く近くを飛ぶと目を覚まし、猫パンチでパウンス（命中で即死）
- **蚊取り線香** … 周囲に煙のダメージゾーン
- **殺虫剤スプレー** … 警戒した人間がたまに噴射する一時的な毒煙
- **ランプの明かり** … 明るい場所はバレやすい
- **時間経過（夜→朝）** … だんだん人間が鋭くなる

---

## 🧠 ステータス

- 🩸 **満腹** … 時間で減少。ゼロで餓死。吸血で回復
- ⚡ **スタミナ** … 飛行・ダッシュで消費。止まって休むと回復
- ❤️ **HP** … 叩かれる・煙で減少
- **警戒度** … 人間/ネコがどれだけ気づいているか

## 🚀 開発

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # dist/ に本番ビルド
npm run preview  # ビルド結果のプレビュー
```

### 構成

```
index.html        UI/HUD/オーバーレイ
src/
  main.js         ゲームループ・状態管理・カメラ・スコア
  world.js        3D部屋・家具・ライト・当たり判定/休憩面
  human.js        人間NPC（徘徊/気づき/叩くAI・吸血スポット）
  mosquito.js     プレイヤーの蚊（モデル・飛行物理・見た目）
  cat.js          ネコ捕食者
  hazards.js      蚊取り線香/スプレー・ミッション管理
  input.js        PC＋タッチ入力の統合
  audio.js        WebAudioによる羽音・効果音（手続き生成）
  ui.js           HUD/ミニマップ/エフェクト/画面遷移
  style.css       スタイル全般
```

`?autostart=1` を付けると自動でゲーム開始（動作確認用）。
